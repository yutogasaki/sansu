# Archive

## Purpose

This folder holds documentation that is no longer active but still worth keeping for reference.

## What Goes Here

- Old status snapshots
- Retired task files
- Historical investigation notes that should not stay in hot docs

## What Must Not Go Here

- Current source-of-truth docs
- Active task files
- Durable memory that still guides present decisions

## Suggested Structure

- `archive/status/`
- `archive/tasks/`
- `archive/notes/`

Create subfolders only when the first archived file actually exists.
