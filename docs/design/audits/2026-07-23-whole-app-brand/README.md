# Whole-app brand audit — 2026-07-23

## Target

- App: local Sansu PWA
- Display name target: `ポッコのふしぎずかん`
- Viewport: 390 × 844
- Benchmark: accepted `dig-pop-painted-v2` exploration art and the yellow leaf-hat companion

## Before evidence

| Flow | Screenshot | Finding |
|---|---|---|
| Onboarding | `01-onboarding-current.png` | Generic sprout, pale glass, and quiet-learning copy contradict the game-first product. |

## Baseline verdict

- Visual appeal: **HOLD** — one strong flow cannot compensate for three mixed visual systems.
- Silent comprehension/safety: **HOLD** — onboarding communicates a quiet study utility, while launch opens a game.
- Runtime integrity: **HOLD** — the development debugger is visible in the normal Study surface.

## After verdict

- Visual appeal: **PASS for the brand-foundation release** — the whole app now shares a recognizable actor, palette, ink, paper, and action vocabulary. Painted Explore art remains the peak register rather than being diluted into utilities.
- Silent comprehension/safety: **PASS** — the first screen says `といて、ほって、みつけよう`; no invented game rule must be understood before starting.
- Runtime integrity: **PASS pending release verification** — route structure and keypad density are unchanged, the persistent footer now points directly to Explore, and the debugger is opt-in only.

Remaining limitation: the new name and mark still require real child/parent recognition testing and formal trademark clearance before public launch.

## Name loop
