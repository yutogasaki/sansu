## What this containment fixes

  to an unrelated painted renderer.
- The regular flower sequence uses one horizontal cyan/ochre stage, one Pokko,
  one five-petal flower, four drops, one road, and one seated body gag.
- The mobile answer shelf no longer overlays the equation, visual quantity hint,
  or TenKey at 390×844 or the 390×700 short-browser viewport.
- The HUD labels the changing `12 / 12` value as `ひかり`; it is not a question
  counter.
- A pending ordinary problem can be left safely with the top-left back arrow;
  failed persistence keeps the active run available for retry.
