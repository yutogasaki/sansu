# Task Archive

## Purpose

This folder holds task or status material that is no longer active but still useful for reference.

## What Goes Here

- Retired task breakdowns
- Old status snapshots moved out of hot docs
- Supporting notes that should not stay in `docs/tasks/active/` or backlog files

## What Must Not Go Here

- Current backlog priorities
- Active execution files
- Durable project truth that belongs in specs, runbooks, or wiki docs
